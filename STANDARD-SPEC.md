# 即時交易台 — 標準規格 (Standard Spec)

> 本規格由 `/learn` 研究產出的功能藍本固化而成（來源見
> `.claude/learn/digests/stock-website-features-and-chart-operations.md`），以主流股票
> 看盤/交易平台（TradingView、Futu/moomoo、富邦/元大）與開源圖表函式庫為標竿。
> 這是本交易台的**驗收基準**：每條需求帶 ID 與優先級，`/review` 以此做符合度（缺口）分析。

**優先級**：`P0` = 一個可信交易台的核心必備；`P1` = 專業深度；`P2` = 進階/加分。
**狀態標記**（由 /review 填寫）：`✓ 已實作` / `◐ 部分` / `✗ 缺`。

**全域不可變規則**：紅漲綠跌（漲=紅 `--up`、跌=綠 `--down`）；價格 TWD、量以「張」（1 張=1000 股）；台股盤別 09:00–13:30 TPE；外部資料來源 TWSE MIS（無金鑰）。

## G. 平台化與品質 (Platform & Quality) — `PQ`

> 由 UX 專家審查（2026-06-23，39 項發現）新增的橫切需求；修正「展示型 demo」與「可信股票網站」之間的品質落差。

| ID | 需求 | 優先級 |
|----|------|--------|
| PQ-1 | 台股 **tick size / 顯示精度依價格分級表動態決定**（顯示小數位 + 下單 step；取代固定 2 位與寫死 0.05） | **P0** |
| PQ-2 | **報價新鮮度**：as-of 資料時間戳 + stale/延遲/收盤 視覺降級；頂列**牆鐘與資料時間分離**（斷線時鐘不凍結） | **P0** |
| PQ-3 | **下單費用模型**：手續費（0.1425%×折扣，最低 20 元）+ 賣出證交稅（0.3%）估算 + 可用購買力檢查（資料化、非恆 0） | P1 |
| PQ-4 | **載入/錯誤/空/斷線 狀態明確區分**（取代 useApi 吞錯回空陣列；loading≠empty≠error） | P1 |
| PQ-5 | **無障礙**：關鍵狀態 aria-live 播報、語意化清單（watchlist）、input label、focus-visible、SVG/盤口 aria；方向不單靠顏色（箭頭+正負號） | P1 |
| PQ-6 | **深色主題對比達 WCAG AA**（--text-3 提亮等；買賣鈕文字對比一致） | P1 |
| PQ-7 | **個股深連結路由** `/stock/:symbol`（symbol 為導覽一等公民、可前進後退/書籤） | P2 |
| PQ-8 | **部署設定化**：連線端點/CORS/埠號由環境變數注入；wss 隨頁面協定；無 '*'、無寫死 localhost | P2 |
